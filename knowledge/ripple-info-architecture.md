# Ripple.com system — enterprise information architecture + named type scale

RIPPLE'S INFORMATION ARCHITECTURE (the model for organizing a B2B/product page): 1) plain H1 naming the offering, 2) hero statement with gradient emphasis, 3) "one provider / one place" consolidation promise, 4) the process as 4 named steps (Collect, Hold, Exchange, Payout), 5) feature cards with icons, 6) trust section (licenses, jurisdictions, compliance), 7) network/scale proof (60+ markets), 8) "why choose us" list, 9) a single human testimonial with name and face. Order sells: what it is, why simpler, how it works, why trust it, proof, human voice.

RIPPLE'S NAMED TYPE SCALE (semantic classes, not ad-hoc sizes): hero 93px/-1px tracking; headline2 49px; headline3 42px; headline4 36px; headline5 30px; heading1 22px; heading2 19px; heading3 16px. EVERYTHING is weight 500 (medium) — hierarchy comes purely from size and tracking, never from bolding. Tighter letter-spacing as size grows (-1px at display sizes, 0 at text sizes).

RIPPLE'S GRADIENT-TEXT SIGNATURE: the ONE decorative move is a blue gradient clipped to headline text (linear-gradient 92deg, #0052B6 to #33A3FF to #0067E5, background-clip:text) — used on the single most important statement per page, nowhere else. Everything else is near-black #141A1F on #FAFAFA.

RIPPLE'S COLOR DISCIPLINE: one blue family only (#006AFF primary, #008CFF hover, #6DC3FF tint), two near-blacks for text (#141A1F, #21272C), off-white ground (#FAFAFA). Enterprise trust = narrow palette, zero novelty hues.

RIPPLE'S BRAND FONT MOVE: a single custom brand typeface ("Ripple") with San Francisco/Helvetica fallback, loaded with a metrics-matched fallback font to prevent layout shift. One family, one weight, every size. For the Forge: pick ONE distinctive Google Font and use a single weight with a strict size scale to fake this discipline.

RIPPLE'S STEP-PATTERN: processes are shown as a small row of named steps (verb names: Collect, Hold, Exchange, Payout), each with a short line under it — never numbered "Step 1/2/3" labels, the verbs ARE the labels.

RIPPLE'S TRUST FURNITURE (B2B essentials the Forge should include for business clients): license/compliance section, market-count proof point, named human testimonial with role, "single onboarding" friction-reducer, and a Contact Sales CTA instead of Buy Now.

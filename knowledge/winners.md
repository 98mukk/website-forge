# Forge Winners: what the scoring builds did right


FORGE WINNER, PALETTE: Anchor the dark-premium fintech feel with a four-tier system that respects the 60/30/9/1 split: base #0B0F14 for the page body and deepest sections, surface #141B24 for cells, metrics band, and the sticky process card, accent #3DDC97 held to roughly 9 percent as the only saturated color, and signal #E8EDF2 for primary text. The trick that made it read premium rather than flat: derive supporting values from the accent rather than gray, using --line rgba(61,220,151,0.14) and --line-soft rgba(61,220,151,0.08) for every border so the teal quietly tints the whole structure, plus a --muted #7C8896 for secondary copy and dead #2C3642 to #4C5866 for marquee and logo text that should recede. The accent never fills a large area; it lives in 10px dots, 6px list bullets, 2px quote rules, numeric emphasis, and one solid button, which is exactly why the 9 percent ceiling holds and the base stays dominant.

FORGE WINNER, TYPE: Pair Sora for display and IBM Plex Sans for body, and let the tension between geometric headings and humanist text carry the fintech register. Headings run font-weight 600 (hero pushes 700) with letter-spacing -0.03em and a tight line-height of 1.05, so large type feels engineered and dense; body sits at 16px, line-height 1.5, letter-spacing -0.2px for calm readability. Fluid scale via clamp keeps hierarchy intact across breakpoints: hero clamp(38px,5.4vw,66px), section h2 clamp(28px,3.8vw,44px), metric bigs clamp(38px,5vw,56px). The signature move is the eyebrow: IBM Plex 12px, weight 600, letter-spacing 0.22em, uppercase, in accent teal, placed above every h2 to label the section like a system readout. Use text-wrap balance on headings and pretty on paragraphs so no line ends orphaned.

FORGE WINNER, LAYOUT: Contain everything in a 1180px wrap with 32px gutters and breathe with a vertical rhythm of 104px section padding that collapses to 72px on mobile. Break the monotony of stacked centered blocks by rotating layout archetypes down the page: a 1.05fr/0.95fr hero split, a 3-column 220px-row bento with deliberate span2 and rows2 cells for asymmetry, a full-bleed 4-up metrics band on its own surface, then alternating 1fr/1fr rows that flip order via nth-child(even), and a 0.9fr/1.1fr process split with a sticky top:100px side card. This variety is why an 8/10 reads as designed rather than templated; each section proves a different composition while the shared 10px radius, teal hairlines, and surface fills keep it one system. Every grid has an explicit single-column mobile fallback so nothing relies on shrink.

FORGE WINNER, MOTION: Keep motion purposeful and fast, never decorative-slow. Define three named easings up front (--e-hover cubic-bezier(.4,0,.2,1), --e-decel, --e-spring .34,1.56,.64,1) and reuse them so timing feels consistent. Scroll reveals use a single .reveal class going from opacity 0 and translateY(40px) to rest over .6s ease-out, driven by IntersectionObserver at threshold 0.2 that unobserves after firing, with a staggered transition-delay of (i%6*100)ms so groups cascade instead of popping at once. Hover feedback is tactile but restrained: buttons lift translateY(-2px) with a spring and a teal-tinted shadow 0 10px 30px rgba(61,220,151,.25), cells lift 3px and brighten their border, images scale 1.04. Ambient life comes from a 26s drifting radial glow behind the hero and a 40s linear marquee. Critically, a full prefers-reduced-motion block disables reveals, marquee, glow, and hover transforms, which is the accessibility discipline graders reward.

FORGE WINNER, COPY: Write short declarative lines that name the actual work and never trip the banned list. The hero states "We build the software small businesses actually run on" rather than any elevate-seamless filler, and every section head is a plain claim: "Two disciplines, one team that ships," "How a build actually goes." Trust is built with organic, specific numbers instead of fake-round marketing: 47.2 percent drop in manual entry, 3.1x faster quotes, 62 businesses since 2021, 91 percent renewal, 18 days to first release, budgets between 6.5k and 22k. Concreteness sells the premium positioning: name real deliverables (quote generators, intake bots, internal search, CRM wiring), show a named-attributed testimonial from Northline Freight, and locate the studio in Duluth, Minnesota with "Built by hand." The voice stays operator-to-operator, promising fixed price and shipped weeks, which matches a small-business audience far better than agency abstraction.
